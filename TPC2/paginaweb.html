ABSTRACT:

Um DTD e uma definição de tipo de documento. Um aplicativo pode usar um DTD para verificar se os dados XML sao validos. Do ponto de vista de um DTD todos os documentos XML são compostos pelos seguintes blocos de construcao: Elementos, Atributos, Entidades PCDATA e CDATA.

Os elementos sao os principais componentes dos documentos XML e HTML.

Os atributos fornecem informações extras sobre os elementos. Estes são sempre colocados dentro de uma tag de abertura de um elemento. Os atributos vêm sempre em pares nome/valor.
